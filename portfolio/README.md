# Shwetansh Singh — Data Analyst Portfolio

A modern, dark-themed portfolio website for a Data Analyst, built with pure HTML, CSS, and JavaScript.

## 🚀 Live Demo
> Deploy on GitHub Pages — link will appear here after setup.

## 📁 Project Structure

```
portfolio/
├── index.html          ← Main HTML file
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← All JavaScript (cursor, canvas, animations)
├── assets/
│   └── images/
│       └── profile-placeholder.svg   ← Replace with your photo
└── README.md
```

## ✨ Features

- Custom animated cursor
- Particle canvas background
- Loading screen with progress bar
- Smooth scroll reveal animations
- Live IST clock in navbar
- Skill bars with animation
- Animated stat counters
- Responsive mobile menu
- Contact form (visual)

## 🛠️ How to Run Locally

Just open `index.html` in your browser — no server needed!

## 🌐 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `portfolio`)
2. Upload all files maintaining the folder structure
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save** — your site will be live at:
   `https://yourusername.github.io/portfolio/`

## 📸 Add Your Photo

Replace `assets/images/profile-placeholder.svg` with your actual photo:
- Recommended size: **400×533px** (3:4 ratio)
- Name it `profile.jpg` or `profile.png`
- Update the `about-img-box` section in `index.html` to use:
  ```html
  <img src="assets/images/profile.jpg" alt="Shwetansh Singh" style="width:100%;height:100%;object-fit:cover;"/>
  ```
